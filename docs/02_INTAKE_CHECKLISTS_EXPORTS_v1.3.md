## B. Approval Phrases (Mandatory)

Use **only** these approvals:

- `APPROVED – Brand Lock`
- `APPROVED – Master Logo`
- `APPROVED – {asset_name}`

If user says “no edits”, export immediately.

## C. Export Checklist

### Always
- `/logo/master.png`
- watermark status verified
- `/ops/` scripts included
- `README.txt` completed

### Starter
- 1 seasonal logo

### Pro
- 7 seasonals
- 3 templates

### Premium
- dark-mode logo
- 7 seasonals
- 6 templates
- showcase poster
- 1 motion prompt pack

### Elite
- all Premium assets
- motion prompts for all assets
